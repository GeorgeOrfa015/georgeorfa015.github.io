2D Platformer Game

This project was made with a tutorial from Chris Courses on YouTube.
It can be found here.
This website isn't mobile compatible, sorry!

Controls
A: Left
D: Right
W: Jump/fly
When you're on a platform, the square will jump.
When airborne, the square will fly instead.
When flying is enabled, the square will become yellow.
When it lands, fly mode will be disabled, and the square will turn back into red.
